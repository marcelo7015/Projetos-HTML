Como desenvolvedores web, somos responsáveis ​​por moldar as experiências de vidas on-line dos usuários. Ao fazer escolhas éticas, centradas no usuário, criamos uma web melhor para todos. A série Ethical Web Development visa analisar as questões éticas do desenvolvimento web. Com isso em mente, tentei dividir as questões éticas da web desenvolvimento em quatro princípios fundamentais: 1. As aplicações da Web devem funcionar para todos. 2. As aplicações da Web devem funcionar em todos os lugares. 3. Os aplicativos da Web devem respeitar a privacidade e a segurança de um usuário. 4. Os desenvolvedores da Web devem ser atentos aos seus pares. Os três primeiros são todos sobre a tomada de decisões éticas para os usuários de nossos sites e aplicativos. Quando criamos aplicativos da web, estamos tomando decisões para os outros, muitas vezes inconscientemente para esses usuários. O quarto princípio diz respeito à forma como interagimos com os outros no nosso indústria.
